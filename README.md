# Minesweeper
## Description:
In this classic strategy game, the user races the clock to uncover a gameboard of tiles while avoiding randomly placed mines.

Our version of the game has three levels:
|Level |Board Size |Number of Bombs |
|:----:|:---------:|:--------------:|
|EASY  |8*8        |10              |
|MEDIUM|16*16      |40              |
|HARD  |24*24      |90              |
